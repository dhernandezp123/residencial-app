# ResidentPass — Branding Assets

Estructura de assets de marca. Todos los archivos aquí son estáticos y se sirven directamente desde `/branding/...`.

---

## branding/logos/

Versiones del logotipo completo (símbolo + texto "ResidentPass").

| Archivo | Uso |
|---|---|
| `residentpass-lockup.svg` | Logo principal sobre fondos claros |
| `residentpass-lockup-reverse.svg` | Logo para fondos oscuros o verdes |
| `residentpass-symbol.svg` | Solo el símbolo (escudo), sin texto |
| `residentpass-symbol-mono.svg` | Símbolo en un solo color (negro o blanco) |
| `residentpass-symbol-white.svg` | Símbolo en blanco puro, para fondos de color |

---

## branding/icons/

Íconos de la app para PWA, tiendas y dispositivos.

| Archivo | Uso |
|---|---|
| `residentpass-icon.svg` | Ícono vectorial maestro |
| `icon-192.png` | PWA manifest — tamaño 192×192 |
| `icon-512.png` | PWA manifest — tamaño 512×512 |
| `icon-maskable-512.png` | PWA maskable icon — 512×512 con safe area |
| `apple-touch-icon.png` | iOS "Agregar a inicio" — 180×180 |

---

## branding/favicons/

Favicons para navegadores de escritorio.

| Archivo | Uso |
|---|---|
| `favicon.ico` | Favicon legacy para todos los navegadores |
| `favicon-32.png` | Favicon PNG 32×32 |
| `favicon-16.png` | Favicon PNG 16×16 |

---

## branding/social/

Imágenes para Open Graph y redes sociales.

| Archivo | Uso |
|---|---|
| `social-card.png` | Imagen de compartir genérica (1200×630) |
| `og-image.png` | Open Graph image para `<meta og:image>` (1200×630) |

---

## templates/

Plantillas de imagen usadas por el sistema para generar tarjetas.

| Archivo | Uso |
|---|---|
| `visit-card-template.png` | Plantilla de fondo para la tarjeta QR de visita compartible |

---

## images/

Imágenes generales de la aplicación (capturas de pantalla, ilustraciones, etc.).
No usar para assets de marca — usar `branding/` para eso.
